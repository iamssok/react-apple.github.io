# [인터랙션 반응형 웹](https://clonecoding-by-react.vercel.app/)
(2024. 07 ~ 09)

기존에 `HTML5/CSS3`, `JavaScript`, `jQuery`를 사용하여 'apple.com/kr' 사이트를 클론 코딩한 인터랙션 반응형 웹사이트를 `React` 컴포넌트 기반 개발 방식으로 리팩토링하였습니다.

- [x] 그 외 필요한 애니메이션

### Home.jsx
